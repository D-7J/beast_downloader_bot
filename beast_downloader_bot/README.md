## 🎯 معرفی

Persian Downloader Bot یک ربات تلگرام حرفه‌ای برای دانلود ویدیو و صوت از بیش از 50 پلتفرم مختلف است. این بات با استفاده از جدیدترین تکنولوژی‌ها طراحی شده و قابلیت‌های منحصر به فردی برای کاربران ایرانی دارد.

### چرا این بات؟
- 🚀 **سرعت بالا**: استفاده از معماری async و صف‌های هوشمند
- 🌐 **پشتیبانی گسترده**: بیش از 50 سایت مختلف
- 💎 **سیستم اشتراک**: مدل درآمدی پایدار
- 🔒 **امنیت بالا**: رعایت حریم خصوصی کاربران
- 📊 **پنل مدیریت**: کنترل کامل بر بات

## 🚀 نصب و راه‌اندازی

### نصب سریع با Docker
```bash
# کلون کردن پروژه
git clone https://github.com/yourusername/persian-downloader-bot.git
cd persian-downloader-bot

# تنظیم متغیرهای محیطی
cp .env.docker .env
nano .env  # ویرایش توکن بات و تنظیمات

# اجرا با Docker Compose
docker-compose up -d
```

### نصب دستی
```bash
# نصب پیش‌نیازها
sudo apt update
sudo apt install python3.11 ffmpeg mongodb redis

# ایجاد محیط مجازی
python3.11 -m venv venv
source venv/bin/activate

# نصب وابستگی‌ها
pip install -r requirements.txt

# اجرای بات
python -m src.bot
```

برای راهنمای کامل نصب، [مستندات deployment](DEPLOYMENT.md) را مطالعه کنید.

## 💰 مدل کسب درآمد

### پلن‌های اشتراک
| پلن | قیمت ماهانه | ویژگی‌ها |
|-----|------------|----------|
| رایگان | 0 تومان | 5 دانلود روزانه، حداکثر 50MB |
| برنزی | 50,000 تومان | 50 دانلود روزانه، حداکثر 200MB |
| نقره‌ای | 100,000 تومان | 150 دانلود روزانه، حداکثر 500MB |
| طلایی | 200,000 تومان | دانلود نامحدود، حداکثر 2GB |

### درآمد تخمینی
- 1000 کاربر فعال: 10-20 میلیون تومان/ماه
- 5000 کاربر فعال: 50-100 میلیون تومان/ماه
- 10000 کاربر فعال: 100-200 میلیون تومان/ماه

## 🛠 تکنولوژی‌ها

### Backend
- **Python 3.11**: زبان اصلی
- **python-telegram-bot**: کتابخانه تلگرام
- **yt-dlp**: موتور دانلود
- **Celery**: صف وظایف
- **AsyncIO**: برنامه‌نویسی ناهمزمان

### Database
- **MongoDB**: دیتابیس اصلی
- **Redis**: کش و صف

### DevOps
- **Docker**: کانتینرسازی
- **Docker Compose**: مدیریت سرویس‌ها
- **Nginx**: وب سرور
- **GitHub Actions**: CI/CD

### Monitoring
- **Flower**: مانیتورینگ Celery
- **Prometheus**: جمع‌آوری متریک
- **Grafana**: نمایش داشبورد
